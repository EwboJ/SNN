### 动作分布 (基于 left1_bag1 样本推断)

每个 run 的典型动作序列为：Stop(起步) → Forward(直行) → Left/Right(转弯) → Forward(出弯)

3 类映射: **Left / Straight(Forward+Stop) / Right**

由于每个 left_* run 只包含 Forward + Left，每个 right_* run 只包含 Forward + Right，预计：
- **Left 类**: 来自 left1-4 的 28 个 run (~40% 帧为 Left 动作)
- **Right 类**: 来自 right1-4 的 28 个 run (~40% 帧为 Right 动作)
- **Straight 类**: 所有 run 的 Forward+Stop 部分 (~60% 帧)

**预估比例: Straight ~60%, Left ~20%, Right ~20%**

> [!IMPORTANT]
> 类别仍不平衡 (Straight 占多数)，需要 `weighted_sampler`

### Step 2: 划分 train/test

按"走廊拐角"做分组划分（而非随机打乱），这样 test set 能测试泛化到未见过的路口：

**推荐划分策略 A（按走廊泛化）:**
- Train: 走廊1,2,3 的所有 runs (42 个 run, ~9,645 帧)
- Test: 走廊4 的所有 runs (14 个 run, ~3,239 帧)
- 比例: 75% / 25%

**推荐划分策略 B（按采集次数）:**
- Train: 每个走廊的 bag1-5 (40 个 run)
- Test: 每个走廊的 bag6-7 (16 个 run)
- 比例: 71% / 29%

> [!IMPORTANT]
> **策略 A（按走廊）泛化测试更有价值**：如果在走廊4上准确率也高，说明网络学到了通用转弯特征而非记忆特定场景。
> **策略 B（按采集次数）**数据分布更均匀，适合论文里报告的准确率数字。

**执行命令（策略 B 示例）:**

```bash
python split_corridor_runs.py \
    --src_root ./data/corridor_all \
    --dst_root ./data/corridor \
    --test_ratio 0.25 \
    --seed 42 --copy_mode copy
```

如果使用策略 A（按走廊划分），需要手动操作或修改 split 脚本。我可以帮你添加按走廊分组划分的功能。

### Step 3: 训练实验矩阵

#### 实验 1: 基线 — 3类离散 + Rate 编码 (最稳定，先跑通)

```bash
python train.py \
    --dataset corridor \
    --corridor_root ./data/corridor \
    --mode discrete --action_set 3 \
    --encoding rate -T 4 \
    --neuron_type APLIF --residual_mode ADD \
    --class_balance weighted_sampler \
    --spike_lambda 0.001 \
    -b 32 -epochs 80 -lr 0.001 \
    -enable_tensorboard --seed 42
```

> 关键参数解释:
> - `-T 4` : 数据量足够后可用 T=4（更快）或 T=8（更精确）
> - `-b 32` : 12K+ 帧可以用较大 batch
> - `--class_balance weighted_sampler` : 解决 Straight 过多问题
> - `--spike_lambda 0.001` : spike rate 正则，抑制过度发放

#### 实验 2: FrameDiff 帧差编码 + 序列训练

```bash
python train.py \
    --dataset corridor \
    --corridor_root ./data/corridor \
    --mode discrete --action_set 3 \
    --encoding framediff -T 1 \
    --seq_len 8 --stride 2 \
    --class_balance weighted_sampler \
    --spike_lambda 0.001 \
    -b 16 -epochs 80 -lr 0.001 \
    -enable_tensorboard --seed 42
```

> 注意:
> - `--encoding framediff -T 1`: 帧差编码，SNN 状态跨帧保留
> - `--seq_len 8 --stride 2`: 8帧序列，步长2
> - `-b 16`: 序列模式显存占用较多，batch 减半

#### 实验 3: 消融 — 不同神经元类型

```bash
# LIF 基线
python train.py --dataset corridor --corridor_root ./data/corridor \
    --mode discrete --action_set 3 -T 4 \
    --neuron_type LIF --residual_mode ADD \
    --class_balance weighted_sampler -b 32 -epochs 80

# PLIF
python train.py --dataset corridor --corridor_root ./data/corridor \
    --mode discrete --action_set 3 -T 4 \
    --neuron_type PLIF --residual_mode ADD \
    --class_balance weighted_sampler -b 32 -epochs 80
```
