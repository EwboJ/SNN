### 图表直接放入 PPT 的方法

#### 方法 A: 直接插入图片 (最快)
所有图片在 `results/baseline_APLIF_ADD_T4/` 目录下，直接拖入 PPT:

```
confusion_matrix.png    → 混淆矩阵页
training_curves.png     → 训练曲线页
spike_analysis.png      → 脉冲分析页
per_class_metrics.png   → 指标明细页
spike_rate_curve.png    → 发放率变化页
```

> 💡 PPT 中插入: 菜单 → 插入 → 图片 → 选择文件

#### 方法 B: 从 metrics.json 提取数字
```python
import json
with open('results/baseline_APLIF_ADD_T4/metrics.json') as f:
    m = json.load(f)
print(f"准确率: {m['accuracy']:.2%}")
print(f"发放率: {m['avg_spike_rate']:.4f}")
print(f"稀疏度: {m['sparsity']:.2%}")
```

#### 方法 C: 用 Python 生成 PPT (python-pptx)
```bash
pip install python-pptx
```
后续可以写脚本自动生成标准 PPT，如果需要我可以协助编写。

#### 方法 D: 消融对比表格
1. 运行完所有消融实验后导出 CSV:
```bash
python eval_corridor_spike_stats.py --batch_compare ^
    checkpoint/.../exp1/best_model.ckpt ^
    checkpoint/.../exp2/best_model.ckpt ^
    --corridor_root ./data/corridor ^
    --export_csv results/ablation_comparison.csv
```
2. 在 Excel 中打开 CSV → 格式化 → 复制粘贴到 PPT
