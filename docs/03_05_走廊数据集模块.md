# 走廊导航数据集模块 — 进展 (2026-03-05)

## 新增模块

### datasets/corridor_dataset.py
走廊导航数据集读取模块，适配 `corridor_export.py` 生成的数据格式。

#### CorridorDataset (单帧模式)
- **mode**: `discrete` (分类) / `regression` (回归)
- **action_set**: `'5'` (Forward/Backward/Left/Right/Stop) / `'3'` (Left/Straight/Right)
- **control_dim**: 1 (angular_z) / 2 (linear_x + angular_z)
- **valid_only**: 过滤 valid=0 的不可信帧
- **多 run 聚合**: 自动扫描 root_dir 下所有 run 子目录
- **return_meta**: 可选返回 timestamp/time_diff_ms 等元数据
- **class_weights / WeightedRandomSampler**: 内置类别不平衡处理

#### CorridorSequenceDataset (序列模式)
- 滑动窗口切出 seq_len 长的帧序列，用于 SNN 跨帧状态学习
- **按 run 边界切分**: 不跨 run 拼接
- **时间连续性检查**: max_gap_ms 参数控制，跳过时间间隔过大的断点

### datasets/__init__.py
包初始化，导出所有公开接口。

### train.py 集成
`build_dataset()` 新增 `corridor` 分支：
```bash
python train.py --dataset corridor -data_dir ./data/corridor
```

## 使用示例

```python
from datasets.corridor_dataset import CorridorDataset, CorridorSequenceDataset

# 离散 3 类 (Left/Straight/Right)
ds = CorridorDataset('data/corridor/', mode='discrete', action_set='3')
sampler = ds.get_weighted_sampler()
loader = DataLoader(ds, batch_size=32, sampler=sampler)

# 回归 angular_z
ds = CorridorDataset('data/corridor/', mode='regression', control_dim=1)

# 序列模式 (SNN T=8)
sds = CorridorSequenceDataset('data/corridor/', seq_len=8, stride=1)
```
