## 架构设计

```
[原始图像]
    ↓
[编码器]
  rate:      直接传入 RGB (3ch)，backbone 内部做 T 步展开
  framediff: 帧差 ReLU(I_t - I_{t-1}) ∥ ReLU(I_{t-1} - I_t) → 6ch
    ↓
[SNN Backbone (SorResNet)]  ← 复用 ADD_ResNet110.py，不修改
  conv1→bn1→sn→layer1→layer2→layer3→avgpool → 128维特征
    ↓
[输出头]
  discrete:   fc → logits [B, num_actions]
  regression: fc → tanh * [v_max, w_max] → [B, control_dim]
```

## 推理接口

| 方法 | 输入 | 场景 |
|---|---|---|
| `forward(x)` | [B,C,H,W] | 单帧训练/推理 |
| `forward_seq(xs)` | [B,L,C,H,W] → [B,L,*] | 序列训练 |
| `step(frame)` | [B,C,H,W] | ROS 在线推理 |
| `get_action(x)` | [1,C,H,W] → int/tensor | 便捷推理 |
| `reset_state()` | — | 重置 SNN 膜电位 + prev_frame |
| `set_prev_frame(f)` | [B,C,H,W] | framediff 初始化 |

## 使用示例

```python
from models.snn_corridor import build_corridor_net

# 离散 3 类 + rate 编码
net = build_corridor_net(head_type='discrete', num_actions=3, T=8)

# 回归 angular_z + framediff 编码
net = build_corridor_net(head_type='regression', control_dim=1,
                         encoding='framediff', T=1)

# 加载 CIFAR-10 预训练骨干
net = build_corridor_net(head_type='discrete', num_actions=3,
    pretrained_backbone='checkpoint/CIFAR-10/APLIF_ADD_T8/best_model.ckpt')
```
