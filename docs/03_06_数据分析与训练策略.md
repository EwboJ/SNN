## 2. 问题诊断

### 2.1 数据量不足
2682 帧对 SNN ResNet-110 (约 1.7M 参数) 来说**严重不足**。

**对策**:
- 使用**强数据增强** (已在 train.py 中配置: RandomHorizontalFlip + ColorJitter)
- **降低模型复杂度**: 考虑用更浅的 backbone 或冻结部分层
- **迁移学习**: 从 CIFAR-10 预训练权重初始化 backbone（conv1 通道不同会跳过，但 layer1/2/3 可迁移）
- **增加采集量**: 建议至少采集到 ~10000 帧(约 30 个 run)

### 2.2 类别不平衡
Straight:Left:Right ≈ 3.4:1:1

**对策**:
- `--class_balance weighted_sampler` (推荐): 对少数类过采样
- `--class_balance class_weight`: CE loss 中加入类别权重

### 2.3 控制量离散化
angular_z 仅有 {0.0, +0.5, -0.5} 三个值 → 回归模式价值有限。

**结论**: **优先用 3 类离散分类模式**。

## 3. 训练策略

### Step 1: 划分 train/test

```bash
python split_corridor_runs.py \
    --src_root ./data \
    --dst_root ./data/corridor \
    --test_ratio 0.2 \
    --seed 42 --copy_mode copy
```

预期结果: 8 train + 2 test (约 2140 train / 540 test 帧)

### Step 2: 实验 A — 离散 3 类基线 (Rate 编码)

```bash
python train.py \
    --dataset corridor \
    --corridor_root ./data/corridor \
    --mode discrete --action_set 3 \
    --encoding rate -T 4 \
    --neuron_type APLIF --residual_mode ADD \
    --class_balance weighted_sampler \
    --spike_lambda 0.001 \
    -b 16 -epochs 100 -lr 0.001 \
    -enable_tensorboard --seed 42
```

> - T=4 而非 8：数据量小，减少计算复杂度
> - batch_size=16：小数据集适当减小 batch
> - lr=0.001：Adam 默认学习率
> - 100 epochs：预计 ~15 分钟

### Step 3: 实验 B — 迁移学习 (从 CIFAR-10 预训练)

如果你已有 CIFAR-10 训练好的 checkpoint:

```bash
python train.py \
    --dataset corridor \
    --corridor_root ./data/corridor \
    --mode discrete --action_set 3 \
    --encoding rate -T 4 \
    --class_balance weighted_sampler \
    -resume checkpoint/CIFAR-10/APLIF_ADD_T8/best_model.ckpt \
    -b 16 -epochs 60 -lr 0.0005 \
    -enable_tensorboard --seed 42
```

> 注意: conv1 通道匹配（都是 3ch RGB），所以所有权重都能加载。
> 需要较小的 lr (0.0005) 避免破坏预训练特征。

### Step 4: 实验 C — FrameDiff 编码

```bash
python train.py \
    --dataset corridor \
    --corridor_root ./data/corridor \
    --mode discrete --action_set 3 \
    --encoding framediff -T 1 \
    --seq_len 8 --stride 2 \
    --class_balance weighted_sampler \
    -b 8 -epochs 100 -lr 0.001 \
    -enable_tensorboard --seed 42
```

> 注意: framediff 需要序列模式 (seq_len>0)，T=1

### Step 5: 评估

```bash
python eval_corridor.py \
    -weights checkpoint/corridor/corridor_discrete_3cls_rate_APLIF_ADD_T4/best_model.ckpt \
    --data_root ./data/corridor/test
```
