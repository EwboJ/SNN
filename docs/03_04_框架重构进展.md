# 框架重构进展 (2026-03-04)

## 概述
将 CIFAR-10 SNN 代码重构为 **研究级可迁移框架**，支持一键消融实验。

## 已完成

### 1. 神经元工厂 (`neuron_model.py`)
- 新增 `build_neuron(neuron_type)` 工厂函数
- 支持: **LIF / ALIF / PLIF / APLIF** 四种神经元
- 返回无参构造器，供网络内部批量创建神经元实例

### 2. 模型参数化 (`ADD_ResNet110.py`)
- `SorResNet.__init__` 新增参数: `T`, `neuron_type`, `residual_mode`, `in_channels`
- `BasicBlock` / `Bottleneck`: 通过 `neuron_builder` 注入神经元, `residual_mode` 选择残差策略
- 时间步: `for t in range(1, self.T)` + `out_spike / self.T` (不再硬编码 8)
- 残差: `standard` (经典 add) vs `ADD` (XOR-like + OR-like 合并)

### 3. 训练脚本升级 (`train_cifar10.py`)
- 新增 `SpikeMonitor` 类:
  - ✅ **可反传 spike rate 正则** (保留计算图的 tensor, 非 `.item()`)
  - ✅ **BaseNode 通用 hook** (支持所有神经元类型, 不再绑定 APLIFNode)
  - ✅ 分层发放率 / 全网 sparsity / spike count 统计
- 实验自动按 `{neuron}_{residual}_T{T}` 分目录保存
- TensorBoard 记录: Acc、Loss、SpikeRate、Sparsity、SpikesPerImage、分层发放率
- Checkpoint 保存 `config` 字段 (neuron_type, residual_mode, T)

### 4. 评测脚本升级 (`test_cifar10.py`)
- 自动从 checkpoint `config` 字段读取配置构建匹配网络
- 输出: 准确率、混淆矩阵、spike_rate、sparsity、FPS、分层发放率
- 自动保存评测报告 `eval_report.txt`

### 5. 新增脉冲统计工具 (`eval_spike_stats.py`)
- 单模型详细分析: 全网/分层发放率、sparsity、spike count、FPS
- 批量对比模式: 多模型对比表 + 分层发放率对比
- 导出 CSV 用于论文作图

### 6. 统一训练入口 (`train.py`)
- 通用入口，支持 cifar10/cifar100/custom 数据集
- 支持 classification / regression 任务类型

## 消融实验命令参考

```bash
# === 神经元消融 ===
python train_cifar10.py --neuron_type APLIF -epochs 151 -enable_tensorboard
python train_cifar10.py --neuron_type ALIF  -epochs 151 -enable_tensorboard
python train_cifar10.py --neuron_type PLIF  -epochs 151 -enable_tensorboard
python train_cifar10.py --neuron_type LIF   -epochs 151 -enable_tensorboard

# === 残差消融 ===
python train_cifar10.py --residual_mode standard -epochs 151 -enable_tensorboard
python train_cifar10.py --residual_mode ADD      -epochs 151 -enable_tensorboard

# === 时间步消融 ===
python train_cifar10.py -T 4  -epochs 151 -enable_tensorboard
python train_cifar10.py -T 8  -epochs 151 -enable_tensorboard
python train_cifar10.py -T 16 -epochs 151 -enable_tensorboard

# === 评测 ===
python test_cifar10.py -weights checkpoint/CIFAR-10/APLIF_ADD_T8/best_model.ckpt --save_cm
python eval_spike_stats.py --batch_compare \
    checkpoint/CIFAR-10/APLIF_ADD_T8/best_model.ckpt \
    checkpoint/CIFAR-10/LIF_ADD_T8/best_model.ckpt \
    --export_csv results/ablation.csv
```
