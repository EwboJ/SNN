## 🎯 下一步执行 (按顺序)

### Step 1: 基线训练 (APLIF + ADD + rate + T=4)

```bash
python train.py --dataset corridor --corridor_root ./data/corridor ^
    --mode discrete --action_set 3 --encoding rate -T 4 ^
    --neuron_type APLIF --residual_mode ADD ^
    --class_balance weighted_sampler ^
    --spike_lambda 0.001 -target_rate 0.1 ^
    -b 32 -epochs 80 -lr 0.001 ^
    -lr_scheduler CosALR -T_max 80 ^
    -enable_tensorboard --seed 42 ^
    --final_test
```

预期输出目录: `checkpoint/corridor/corridor_discrete_3cls_rate_APLIF_ADD_T4/`
- `best_model.ckpt` — val 最佳模型
- `final_model.ckpt` — 最终模型
- `final_test_metrics.json` — test 集最终结果
- `runs/` — TensorBoard events

### Step 2: 生成图表

```bash
python scripts/plot_results.py ^
    --ckpt checkpoint/corridor/corridor_discrete_3cls_rate_APLIF_ADD_T4/best_model.ckpt ^
    --data_root ./data/corridor/test ^
    --out_dir results/baseline_APLIF_ADD_T4
```

输出: `confusion_matrix.png`, `training_curves.png`, `spike_analysis.png`, `per_class_metrics.png`

    checkpoint/corridor/corridor_discrete_3cls_rate_APLIF_ADD_T4/best_model.ckpt ^

## 关键提醒

1. **先执行 Step 1 基线，确认能跑通再做消融**
2. 每个实验约 80 epoch × ~200 batch ≈ 20-40 分钟 (取决于 GPU)
3. `--final_test` 确保训练结束时对 test 集做一次独立评估
4. Val 用于调参选模型，Test 只在最终报告结果
