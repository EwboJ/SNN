# 走廊导航离线评估 — 进展 (2026-03-05)

## 新增文件

| 文件 | 说明 |
|---|---|
| eval_corridor.py | 离线评估脚本 (~370行) |

## 功能

### 离散模式 (--mode discrete)
- Accuracy
- Per-class Precision / Recall / F1
- 混淆矩阵 → `confusion_matrix.png` + `confusion_matrix.csv`

### 回归模式 (--mode regression)
- 整体 MAE / RMSE
- 分维度 MAE / RMSE (angular_z / linear_x)
- 预测 vs GT 曲线图 → `curves.png`
- 散点图 → `scatter.png`

### SNN 指标 (两种模式通用)
- avg_spike_rate / sparsity
- spikes_per_frame (能耗 proxy)
- 分层发放率 (layer_rates)
- 推理耗时: avg_forward_ms / FPS (含 torch.cuda.synchronize)

### 输出结构
```
results/<exp_name>/
├── metrics.json     # 全部数值指标
├── config.json      # 命令行 args + checkpoint config (可复现)
├── confusion_matrix.csv   # (discrete)
├── confusion_matrix.png   # (discrete)
├── curves.png             # (regression)
└── scatter.png            # (regression)
```

## 使用示例

```bash
# 离散分类评估
python eval_corridor.py \
    -weights checkpoint/corridor/best_model.ckpt \
    --data_root ./data/corridor/test

# 回归评估
python eval_corridor.py \
    -weights checkpoint/corridor/best_model.ckpt \
    --data_root ./data/corridor/test \
    --mode regression --control_dim 1

# 手动指定配置 (当 checkpoint 无 config 时)
python eval_corridor.py \
    -weights best.pth \
    --data_root ./data/corridor/test \
    --mode discrete --action_set 3 -T 8 \
    --neuron_type APLIF --residual_mode ADD
```
