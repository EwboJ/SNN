## 二、当前重点：阶段 2 — SNN 模型架构与训练框架

### 2.1 研究框架建设（已完成 ✅）

> **目标**：构建可配置、可复现、可消融的 SNN 研究框架——同一套代码训练 CIFAR / 公共数据 / 走廊数据

**核心成果：**

- **自适应脉冲神经元模型 (APLIF)**
  - 可学习膜电位时间常数 τ（Parametric）
  - 指数抑制项抑制过度发放（Adaptive）
  - 对比基线：LIF / ALIF / PLIF / APLIF 四种神经元

- **ADD 残差连接**
  - 针对脉冲二值特性设计的 XOR-like + OR-like 残差
  - 对比基线：标准 ResNet 残差 vs ADD 残差

- **可配置研究框架**
  - 一行命令切换：神经元类型 / 残差模式 / 时间步 T
  - 自动实验分目录、TensorBoard 可视化
  - 可反传的 Spike Rate 正则化（保留计算图）

---

### 2.2 框架架构概览

```
┌─────────────────────────────────────────────────┐
│              SNN 可迁移研究框架                    │
├─────────────────────────────────────────────────┤
│  神经元工厂    build_neuron()                     │
│  ├── LIF      (固定 τ, spikingjelly 内置)        │
│  ├── PLIF     (可学习 τ, spikingjelly 内置)      │
│  ├── ALIF     (固定 τ + 指数抑制)                 │
│  └── APLIF    (可学习 τ + 指数抑制) ← 我们的模型   │
├─────────────────────────────────────────────────┤
│  残差策略      residual_mode                      │
│  ├── standard (经典 ResNet: out + identity)       │
│  └── ADD      (XOR-like + OR-like) ← 我们的方法   │
├─────────────────────────────────────────────────┤
│  网络骨干      ADD-ResNet-110 (SorResNet)         │
│  ├── 时间步 T 可配置 (4/8/16)                     │
│  ├── 输入通道可配置 (RGB=3 / DVS=2 / 灰度=1)     │
│  └── 分类 / 回归 任务可切换                        │
├─────────────────────────────────────────────────┤
│  评估工具链                                       │
│  ├── 准确率 + 混淆矩阵                            │
│  ├── 全网/分层 Spike Rate & Sparsity              │
│  ├── 每张图 Spike Count (能耗代理)                 │
│  └── FPS 测量 + CSV 导出                          │
└─────────────────────────────────────────────────┘
```
