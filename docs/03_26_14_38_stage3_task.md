## 完成时间

2026-03-26 14:38

## 修改文件

### `scripts/derive_stage1_datasets.py`

新增 stage3 三阶段分类任务（Approach/Turn/Recover），彻底丢弃 Follow。

**新增常量：**
- `STAGE3_LABELS = {'Approach': 0, 'Turn': 1, 'Recover': 2}`

**新增函数 `derive_stage3()`：**
- 复用 `classify_phase_stage4()` 做阶段判定，丢弃 Follow 和超出范围的帧
- 每阶段独立 max_frames 裁剪，采样策略由 `--stage3_sample_policy` 控制
- Approach 默认 tail（保留靠近 Turn 的尾部），Turn/Recover 默认 uniform
- 返回 raw_counts / derived_counts 用于统计

**注册：**
- `TASK_FUNCS['stage3'] = derive_stage3`
- `TASK_LABEL_NAMES['stage3'] = {0: 'Approach', 1: 'Turn', 2: 'Recover'}`

**新增 argparse 参数（Stage3 参数组）：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `--stage3_pre_turn_ms` | None | 专用 pre_turn_ms |
| `--stage3_recover_ms` | None | 专用 recover_ms |
| `--stage3_turn_margin_ms` | 300 | Turn 边界 margin |
| `--stage3_max_approach_frames` | 0 | Approach 最多帧 |
| `--stage3_max_turn_frames` | 0 | Turn 最多帧 |
| `--stage3_max_recover_frames` | 0 | Recover 最多帧 |
| `--stage3_sample_policy` | tail | 裁剪策略 |

**dataset_summary.json 增加 `stage3` 节：**
- 完整 stage3 参数
- `phase_trim_summary`（每阶段 raw/derived 帧数汇总）
- 每 run 的 raw_counts / derived_counts

**输出目录：** `stage3_v1/`

**使用示例：**
```bash
python scripts/derive_stage1_datasets.py \
    --task stage3 \
    --stage3_max_approach_frames 30 \
    --stage3_max_turn_frames 20 \
    --stage3_max_recover_frames 20
```
