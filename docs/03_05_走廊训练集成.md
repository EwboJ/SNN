# 走廊导航训练集成 — 进展 (2026-03-05)

## 修改文件

| 文件 | 操作 | 说明 |
|---|---|---|
| train.py | REWRITE | 全面集成走廊导航训练，CIFAR 路径不变 |

## 新增 CLI 参数

| 参数 | 默认值 | 说明 |
|---|---|---|
| `--corridor_root` | ./data/corridor | 走廊数据根目录 |
| `--mode` | discrete | discrete / regression |
| `--action_set` | 3 | 5 或 3 (Left/Straight/Right) |
| `--control_dim` | 1 | 1=angular_z, 2=[v,w] |
| `--encoding` | rate | rate / framediff |
| `--seq_len` | 0 | 0=单帧, >0=序列训练 |
| `--stride` | 1 | 序列步长 |
| `--valid_only` | true | 丢弃 valid=0 帧 |
| `--loss_type` | auto | ce/mse_onehot/huber/auto |
| `--class_balance` | none | weighted_sampler/class_weight |
| `--smooth_lambda` | 0.0 | 动作平滑正则 |
| `--spike_lambda` | 0.001 | spike rate 正则 |

## 使用示例

```bash
# CIFAR-10 (完全不变)
python train.py -T 8 -epochs 151 -enable_tensorboard

# 走廊 离散3类 rate编码
python train.py --dataset corridor --corridor_root ./data/corridor \
    --mode discrete --action_set 3 -T 8

# 走廊 回归 framediff编码
python train.py --dataset corridor --corridor_root ./data/corridor \
    --mode regression --control_dim 1 --encoding framediff -T 1

# 走廊 序列训练
python train.py --dataset corridor --corridor_root ./data/corridor \
    --mode discrete --seq_len 8 --stride 2
```
